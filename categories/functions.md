# Функциональные категории

| ID | Название | Описание |
|---|---|---|
| `sales` | Продажи | Лидогенерация, квалификация, CRM, скрипты, follow-up |
| `marketing` | Маркетинг | Контент, SEO, таргет, аналитика, бренд |
| `hr` | HR | Найм, онбординг, оценка, обучение, удержание |
| `finance` | Финансы | Отчётность, прогнозирование, бюджетирование, аудит |
| `operations` | Операции | Процессы, логистика, планирование, контроль качества |
| `customer-service` | Клиентский сервис | Поддержка, чат-боты, обработка обращений, NPS |
| `it` | ИТ | Разработка, DevOps, документация, безопасность |
| `legal` | Юридический | Договоры, комплаенс, риски, шаблоны документов |
| `management` | Управление | Стратегия, совещания, отчёты, принятие решений |
