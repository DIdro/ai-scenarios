## Триггер

Завершилось совещание с записью или транскриптом.

## Вход

Аудиозапись или текстовый транскрипт совещания (+ опционально фото флипчарта/доски).

## Выход

Структурированный список задач с ответственными и дедлайнами, автоматически созданный в трекере.

## Боль

На оперативных совещаниях обсуждается 12–40 вопросов. Часть теряется, часть «зависает» без ответственного. Ручная расшифровка и структурирование протокола занимают 1–3 часа после каждой встречи.

## Решение

ИИ получает транскрипт, извлекает вопросы, решения и ответственных, дедуплицирует с открытыми задачами, автоматически создаёт задачи в трекере с назначением.

## Шаги внедрения

1. **Транскрибация** — подключить сервис к формату встреч (Zoom/Teams/телефон)
2. **Промпт извлечения** — задачи, решения, ответственные, дедлайны
3. **Дедупликация** — сравнение с открытыми задачами в трекере
4. **Интеграция с трекером** — API или n8n для создания задач
5. **Граф зависимостей** — опционально: связи между задачами

## Инструменты

| Инструмент | Роль |

| Яндекс.Трекер / Jira / АтласТрекер | Создание задач |

## Результаты

| Метрика | До | После |
|---|---|---|
| Время на протокол | 1–3 часа | 10–15 минут |
| % встреч с зависшими задачами | ~60% | ~15% |
| Длительность встреч | базовое | −20–30% |

## Подводные камни

- Качество транскрипции падает при плохом звуке или нескольких говорящих одновременно
- Требует дисциплины: встречи нужно записывать
- Автоназначение ответственных работает только если имена совпадают с профилями в трекере
- Первые 2–3 недели нужна ручная проверка
