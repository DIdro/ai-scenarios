## Триггер

Поступил новый тендерный лот — нужно быстро оценить сроки и стоимость.

## Вход

Исходно-технические требования (ИТТ) в PDF/DOC + база выполненных проектов с фактической трудоёмкостью.

## Выход

Структурированный разбор требований + оценка трудоёмкости по аналогам + параметрическая ценовая модель.

## Боль

Анализ одного тендерного лота занимает 80–160 человеко-часов. Ошибки в оценке сроков достигают 20–30%. Пока компания считает — конкуренты уже подали заявку.

## Решение

Система разбирает ИТТ, извлекает ключевые параметры, ищет аналогичные проекты в базе и рассчитывает трудоёмкость по историческим данным. Менеджер задаёт уточняющие вопросы на естественном языке.

## Шаги внедрения

1. **База проектов** — структурировать выполненные проекты с фактической трудоёмкостью
2. **Парсер ИТТ** — извлечение ключевых параметров из документации
3. **Поиск аналогов** — RAG по базе проектов
4. **Параметрическая модель** — расчёт стоимости на основе исторических данных
5. **NLQ-интерфейс** — вопросы к документации на естественном языке
