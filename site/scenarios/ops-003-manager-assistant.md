## Триггер

Руководитель хочет получить информацию о состоянии проектов, задач или метриках — не заходя в несколько систем.

## Вход

Вопрос на естественном языке: «Какие задачи просрочены в отделе?», «Как выглядит выручка за неделю?»

## Выход

Ответ на естественном языке с данными из корпоративных систем: «Просрочено 4 задачи. Ответственный — Иван (3), Мария (1). Самая старая — с 15 марта».

## Боль

Топ-менеджер тратит часы на сбор информации перед совещаниями: статусы проектов, просроченные задачи, финансовые показатели — всё в разных системах. Подготовка к планёрке = ручной мониторинг 4–5 инструментов.

## Решение

Чат-бот в Telegram подключён ко всем корпоративным системам. Принимает вопросы на естественном языке, определяет нужный источник, делает запрос, синтезирует ответ.

## Шаги внедрения

1. **Инвентаризация источников** — 3–5 ключевых систем (трекер, CRM, финансы, HR)
2. **API-коннекторы** — подключение через n8n или прямые интеграции
3. **Системный промпт** — описание бизнеса, терминов, структуры для LLM
4. **NLQ-роутер** — определение нужного источника по вопросу, запрос, синтез ответа
5. **Telegram-бот** — интерфейс для доступа с телефона

## Подводные камни

- Требует надёжных API у всех источников
- Безопасность: бот имеет доступ к чувствительным данным, нужна авторизация
- Качество ответов зависит от качества данных в системах
