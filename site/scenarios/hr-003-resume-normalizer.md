## Триггер

Получено резюме кандидата в произвольном формате — нужно передать нанимающему менеджеру или заказчику.

## Вход

Резюме в любом формате: PDF, DOC, RTF, ссылка на Google Doc или HH.ru.

## Выход

Унифицированный документ по корпоративному шаблону в Google Docs — готов к передаче без ручной обработки.

## Боль

HR тратит 15–30 минут на приведение каждого резюме к единому виду. При потоке 20–50 резюме в неделю это становится основной работой вместо реального рекрутинга.

## Решение

Бот принимает резюме (файл или ссылка), извлекает структурированные данные и автоматически создаёт унифицированный документ по шаблону. HR получает готовый файл за секунды.

## Шаги внедрения

1. **Шаблон** — зафиксировать стандарт: поля, порядок, формат
2. **Приёмник** — Telegram-бот или email принимает файлы и ссылки
3. **Извлечение** — ИИ структурирует данные по заданным полям
4. **Генерация** — автосоздание Google Doc по шаблону
5. **Доставка** — готовый документ возвращается HR

## Подводные камни

- Резюме с нестандартной вёрсткой (таблицы, колонки) парсятся с ошибками
- Сканированные PDF без текстового слоя требуют OCR
- Нужна проверка: ИИ иногда «галлюцинирует» поля которых нет в оригинале
