## Что болит

Топ-менеджер тратит часы на сбор информации перед совещаниями: статусы
проектов, просроченные задачи, финансовые показатели — всё в разных
системах. Подготовка к планёрке превращается в ручной мониторинг
4–5 инструментов.

## Как это работает

Персональный чат-бот, подключённый ко всем корпоративным системам.
Отвечает на вопросы на естественном языке: «Какие проекты просрочены?»,
«Как выглядит загрузка отдела?», «Сделай сводку по выручке за неделю».
Доступен через Telegram — то есть с телефона, между встречами.

<Steps>
1. **Инвентаризация источников.** Определяем 3–5 ключевых систем —
   трекер, CRM, финансы, HR.
2. **API-коннекторы.** Подключаем каждую систему через n8n или прямые
   интеграции.
3. **Контекстный промпт.** Описываем для LLM бизнес, термины,
   структуру компании, чтобы NLQ-вопросы интерпретировались правильно.
4. **NLQ-интерфейс.** Бот принимает вопрос, определяет нужный источник,
   делает запрос, формирует ответ.
5. **Telegram-обёртка.** Удобный доступ с телефона.
</Steps>

## Подводные камни

<Callout type="warning" title="Безопасность — главный риск">
  Бот имеет доступ к чувствительным данным. Нужна авторизация по
  пользователю и аудит запросов. Не пропускайте этот этап «на потом».
</Callout>

Сценарий требует надёжных API у источников — а у некоторых корпоративных
систем их просто нет, или они кривые. Качество ответов прямо зависит
от качества данных в системах: если в трекере хаос, дашборд не спасёт.
