# AI Scenarios Library 🤖

> База сценариев внедрения ИИ — часть методологии Диалектики.

Практическая библиотека кейсов и сценариев применения искусственного интеллекта в бизнесе. Не теория — реальные задачи, инструменты и результаты.

## Структура репозитория

```
ai-scenarios/
├── scenarios/          # Все сценарии (по одному файлу на сценарий)
├── categories/         # Описания категорий и тегов
├── schema/             # JSON-схема карточки сценария
├── data/               # Агрегированные данные (JSON/CSV) для фронтенда
└── site/               # Исходники сайта (TBD)
```

## Как добавить сценарий

Скопируйте шаблон из `schema/scenario-template.md`, заполните все поля и создайте файл в папке `scenarios/`.

## Категории

**По функции:**
- `sales` — Продажи
- `marketing` — Маркетинг
- `hr` — Управление персоналом
- `finance` — Финансы
- `operations` — Операции и процессы
- `customer-service` — Клиентский сервис
- `it` — ИТ и разработка
- `legal` — Юридический
- `management` — Управление и стратегия

**По отрасли:**
- `retail` — Ритейл
- `manufacturing` — Производство
- `services` — Услуги
- `real-estate` — Недвижимость
- `education` — Образование
- `healthcare` — Здравоохранение
- `fintech` — Финтех
- `it-company` — ИТ-компании

## Статусы сценариев

- `verified` — Проверен в реальном проекте
- `hypothesis` — Гипотеза, не проверена
- `in-progress` — В процессе внедрения
