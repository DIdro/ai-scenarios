## Боль

Топ-менеджер тратит часы на сбор информации перед совещаниями: статусы проектов, просроченные задачи, финансовые показатели — всё в разных системах. Подготовка к планёрке = ручной мониторинг 4–5 инструментов.

## Решение

Персональный чат-бот, подключённый ко всем корпоративным системам. Отвечает на вопросы на естественном языке: «Какие проекты просрочены?», «Как выглядит загрузка отдела?», «Сделай сводку по выручке за неделю». Доступен через Telegram.

## Шаги внедрения

1. **Инвентаризация источников** — определить 3–5 ключевых систем (трекер, CRM, финансы, HR)
2. **API-коннекторы** — подключить каждую систему через n8n или прямые интеграции
3. **Контекстный промпт** — описать бизнес, термины, структуру компании для LLM
4. **NLQ-интерфейс** — бот принимает вопрос, определяет нужный источник, делает запрос, формирует ответ
5. **Telegram-бот** — обёртка для удобного доступа с телефона

## Подводные камни

- Требует надёжных API у всех источников — не у всех систем они есть
- Безопасность: бот имеет доступ к чувствительным данным, нужна авторизация
- Качество ответов зависит от качества данных в системах
- Высокая стоимость разработки интеграций при большом числе источников
